# Python Network Scanner

A Python-based network scanner that identifies open TCP ports using socket programming, demonstrating fundamental cybersecurity, networking, and automation concepts.

## Features

* Scan multiple TCP ports
* Detect open ports on a target host
* Lightweight and easy to use
* Built with Python socket programming

## Technologies Used

* Python
* Socket Programming
* TCP/IP Networking

## Usage

Run the scanner:

```bash
python scanner.py
```

Enter the target IP address when prompted.

## Educational Purpose

This project was created to learn networking fundamentals, TCP/IP communication, socket programming, and cybersecurity concepts.

## Disclaimer

This tool is intended for educational purposes and authorized security testing only.
